Personal Website SMS Notification Service

This is a personal website project used only for personal development and monitoring purposes.

Purpose of SMS Messaging

SMS messages are used solely to send alerts and notifications when abnormal conditions or system issues are detected on the website. No marketing or promotional messages are sent.

User Consent (Opt-in)

Phone numbers are added by the website owner for the purpose of receiving system alerts. The phone number owner has explicitly consented to receive SMS messages related to website monitoring and abnormal activity notifications.

Opt-out

The recipient may opt out of receiving SMS messages at any time by replying STOP.
